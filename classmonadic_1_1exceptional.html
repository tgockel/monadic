Call some action if this instance is not success (the opposite of map).

Parameters
actionis some function which is given an std::exception_ptr and returns a value. It is only called if is_success is false.
Returns
An instance with a copy of the value in this one if is_success is true; an instance with the result of calling action if is_success if false and it returns in success; otherwise, the result will have the exception thrown from action.
