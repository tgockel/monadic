completion_chaining

Perform the next step of the process when the value is delivered in success. The func is only called in the success case (see recover for the failure case).

Template Parameters
FuncR (*)(T&&) or R (*)() if T is void; R can be void
Parameters
funcThis function takes in the successful result of this completion and returns another value. The result of this function will be delivered to the completion_promise that backs the completion returned from this function.
Returns
a completion that will be completed with the result of func.

completion chaining

completion_promise<int> p;
completion<int> c1 = p.get_completion();
completion<double> c2 = c1.map([] (int x) { return double(x); });
completion<std::string> c3 = c2.map([] (double d) { return to_string(d); );
completion<void> c4 = c3.map([] (std::string s) { std::cout << s; });
completion<int> c5 = c4.map([] () { std::cout << std::endl; return 1; });
p.set_value(1);
c5.get();
