inlineexplicit

Create a promise with the given location to store data impl.

The provided completion_data must be uniquely used for this instance and cannot have been used before.

Note
This constructor exists to enable bulk allocation of completion_data instances in non-critical sections of code (as memory allocation can be expensive).
